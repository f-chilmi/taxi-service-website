# Taxi Service Website - Popo's Professional Taxi

A modern, mobile-first website for independent taxi drivers to showcase their services, connect with customers, and build their business online.

## 🚗 Project Overview

This project creates a professional web presence for Popo, an independent taxi driver in Turkey, providing customers with easy access to reliable transportation services through a clean, user-friendly website.

### Key Features
- **Mobile-first responsive design** - Optimized for Turkish mobile users
- **Multi-language support** - Turkish and English interfaces
- **WhatsApp integration** - Primary booking method for Turkish market
- **Transparent pricing** - Clear fare calculator and route pricing
- **Professional presentation** - Build trust with customers
- **Local market focus** - Designed for Turkish taxi service needs

## 🎯 Target Users
- Local residents needing taxi services
- Tourists visiting Turkey
- Airport transfer customers
- Business travelers
- Anyone seeking reliable, personal taxi service

## 🛠️ Technology Stack

### Frontend (Planned)
- **Framework**: React.js with Next.js or Vanilla HTML/CSS/JS
- **Styling**: Tailwind CSS or Bootstrap
- **Icons**: Heroicons or Font Awesome
- **Maps**: Google Maps API integration
- **Languages**: Turkish (primary), English

### Hosting & Deployment
- **Hosting**: Netlify or Vercel (free tier)
- **Domain**: .com or .tr extension
- **Performance**: Optimized for mobile networks in Turkey
- **Analytics**: Google Analytics integration

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- Git
- Text editor (VS Code recommended)

### Installation
```bash
# Clone the repository
git clone [repository-url]
cd taxi-service-website

# Install dependencies (when package.json exists)
npm install

# Start development server (when implemented)
npm run dev
```
